# JMSAutoMapper Codex Guidance Pack

Este pacote contém arquivos de orientação para o Codex trabalhar no repositório JMSAutoMapper.

## Arquivos incluídos

```text
AGENTS.md
docs/governance/CODING-STANDARDS.md
docs/governance/PERFORMANCE-GUIDELINES.md
docs/governance/TESTING-GUIDELINES.md
docs/governance/VERSIONING.md
```

## Como usar

Copie o `AGENTS.md` para a raiz do repositório.

Copie a pasta `docs/governance` para o repositório.

O Codex deve seguir estas regras antes de alterar a solução.
