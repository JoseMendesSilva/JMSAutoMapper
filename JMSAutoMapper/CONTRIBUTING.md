# Guia de Contribuição

Agradecemos seu interesse em contribuir com o JMSAutoMapper!

## Como Contribuir

- **Bugs**: Se encontrar um bug, por favor, abra uma issue detalhando o problema e, se possível, um exemplo de código que o reproduza.
- **Melhorias**: Se você tem uma ideia para uma melhoria, abra uma issue para discussão.
- **Pull Requests**: Pull requests são bem-vindos. Para mudanças maiores, por favor, abra uma issue primeiro para discutir o que você gostaria de mudar.

## Padrões de Código

- Mantenha o estilo de código existente.
- Adicione testes para qualquer nova funcionalidade ou correção de bug.
- Certifique-se de que todos os testes estão passando antes de enviar um pull request.
