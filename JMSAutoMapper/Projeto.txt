yaml

name: Run Tests

      - uses: actions/checkout@v2

Implementar Map<IEnumerable<T>> em JMSAutoMapper
Meu JMSAutoMapper não está aceitando esta chamada ; mapper.Map<IEnumerable<T>>(entities), o JmsAutoMapper ja implementa esta estrutura (mapper.Map<T>(entities)), como posso implementar um método com esta estrutura Map<IEnumerable<T>>(entities)?
Se o JMSAutoMapper já implementa o método mapper.Map<T>(entities) mas não suporta diretamente mapper.Map<IEnumerable<T>>(entities), você pode criar um método de extensão para adicionar essa funcionalidade. Veja como você pode implementar:

1. Criando um método de extensão para o IMapper
csharp
using System;
using System.Collections.Generic;
using System.Linq;

public static class JMSAutoMapperExtensions
{
    public static IEnumerable<TDestination> MapEnumerable<TSource, TDestination>(
        this IMapper mapper, 
        IEnumerable<TSource> source)
    {
        if (source == null)
            throw new ArgumentNullException(nameof(source));

        return source.Select(item => mapper.Map<TDestination>(item));
    }

    // Sobrecarga para manter a assinatura similar ao Map<T>
    public static IEnumerable<TDestination> Map<IEnumerable<TDestination>>(
        this IMapper mapper, 
        IEnumerable<TSource> source)
    {
        return mapper.MapEnumerable<TSource, TDestination>(source);
    }
}
2. Como usar
csharp
// Usando o método de extensão
var mappedList = mapper.MapEnumerable<EntidadeOrigem, EntidadeDestino>(entities);

// Ou usando a sintaxe que você deseja (se optar pela segunda sobrecarga)
var mappedList = mapper.Map<IEnumerable<EntidadeDestino>>(entities);
3. Alternativa (se você puder modificar o JMSAutoMapper)
Se você tiver acesso ao código-fonte do JMSAutoMapper, você pode adicionar diretamente o método:

csharp
public interface IJMSAutoMapper
{
    // Método existente
    T Map<T>(object source);
    
    // Novo método
    IEnumerable<T> MapEnumerable<T>(IEnumerable<object> source);
}

public class JMSAutoMapper : IJMSAutoMapper
{
    private readonly IMapper _mapper;
    
    public JMSAutoMapper(IMapper mapper)
    {
        _mapper = mapper;
    }
    
    public T Map<T>(object source) => _mapper.Map<T>(source);
    
    public IEnumerable<T> MapEnumerable<T>(IEnumerable<object> source)
    {
        return source.Select(item => _mapper.Map<T>(item));
    }
}

Escolha a abordagem que melhor se adapta à sua arquitetura. O método de extensão (opção 1) geralmente é a solução mais limpa sem modificar o código existente.

Poderia reescrever usando o modelo abaixo:
        public T? Map<T>(object? source) where T : class, new()
        {

            return MapObject<T>(source, new HashSet<object>());

Claro! Vou criar uma implementação que segue o mesmo padrão do seu método Map<T>, mas para coleções (IEnumerable<T>), mantendo a mesma estrutura com a verificação de null e usando MapObject<T> internamente.

Implementação do Método para IEnumerable<T>
csharp
using System;
using System.Collections.Generic;
using System.Linq;

public IEnumerable<T>? Map<IEnumerable<T>>(object? source) where T : class, new()
{
    if (source == null)
        return null;

    // Verifica se a fonte é realmente uma coleção
    if (source is not IEnumerable<object> sourceCollection)
        throw new ArgumentException("O objeto de origem deve ser uma coleção.", nameof(source));

    // Mapeia cada item da coleção usando o MapObject<T> existente
    return sourceCollection
        .Select(item => MapObject<T>(item, new HashSet<object>()))
        .ToList(); // Materializa a lista para evitar múltiplas iterações
}
Explicação:
Verificação de null: Assim como no seu método Map<T>, retornamos null se a fonte for null.

Validação do tipo de entrada: Garantimos que o objeto de origem seja uma coleção (IEnumerable<object>).

Mapeamento de cada item: Usamos Select para aplicar MapObject<T> em cada elemento, assim como o método original faz.

Materialização (ToList()): Convertemos o resultado em uma lista para evitar problemas com avaliação tardia (lazy evaluation) do LINQ.

Reutiliza Map<IEnumerable<T>> e converte o resultado em List<T> usando .ToList().

Map<ICollection<T>>

using System;
using System.Collections.Generic;
using System.Linq;

public HashSet<T>? Map<HashSet<T>>(object? source) where T : class, new()
{
    var enumerableResult = Map<IEnumerable<T>>(source);
    return enumerableResult?.ToHashSet(); // .NET 6+ (ou use new HashSet<T>(enumerableResult))
}

public Dictionary<TKey, TValue>? Map<Dictionary<TKey, TValue>>(object? source) 
    where TKey : notnull
    where TValue : class, new()
{
    if (source == null)
        return null;

    if (source is not IEnumerable<KeyValuePair<object, object>> sourceCollection)
        throw new ArgumentException("O objeto de origem deve ser um Dictionary ou coleção de KeyValuePair.", nameof(source));

    var dictionary = new Dictionary<TKey, TValue>();
    foreach (var kvp in sourceCollection)
    {
        var key = MapObject<TKey>(kvp.Key, new HashSet<object>());
        var value = MapObject<TValue>(kvp.Value, new HashSet<object>());
        dictionary.Add(key!, value!);
    }
    return dictionary;
}

public Queue<T>? Map<Queue<T>>(object? source) where T : class, new()
{
    var enumerableResult = Map<IEnumerable<T>>(source);
    return enumerableResult != null ? new Queue<T>(enumerableResult) : null;
}

public Stack<T>? Map<Stack<T>>(object? source) where T : class, new()
{
    var enumerableResult = Map<IEnumerable<T>>(source);

Implementação para Coleções Thread-Safe
1. ConcurrentDictionary<TKey, TValue>
csharp
using System.Collections.Concurrent;

public ConcurrentDictionary<TKey, TValue>? Map<ConcurrentDictionary<TKey, TValue>>(object? source)
    where TKey : notnull
    where TValue : class, new()
{
    if (source == null)
        return null;

    if (source is not IEnumerable<KeyValuePair<object, object>> sourceCollection)
        throw new ArgumentException("O objeto de origem deve ser um Dictionary ou coleção de KeyValuePair.", nameof(source));

    var concurrentDict = new ConcurrentDictionary<TKey, TValue>();
    foreach (var kvp in sourceCollection)
    {
        var key = MapObject<TKey>(kvp.Key, new HashSet<object>());
        var value = MapObject<TValue>(kvp.Value, new HashSet<object>());
        concurrentDict.TryAdd(key!, value!); // Thread-safe
    }
    return concurrentDict;
}
2. BlockingCollection<T>
csharp
public BlockingCollection<T>? Map<BlockingCollection<T>>(object? source, int? boundedCapacity = null)
    where T : class, new()
{
    var enumerableResult = Map<IEnumerable<T>>(source);
    if (enumerableResult == null)
        return null;

    return boundedCapacity.HasValue
        ? new BlockingCollection<T>(new ConcurrentQueue<T>(enumerableResult), boundedCapacity.Value)
        : new BlockingCollection<T>(new ConcurrentQueue<T>(enumerableResult));
}

Implementação em JMSAutoMapper
csharp
public class JMSAutoMapper : IMapper
{
    // ... (implementações anteriores)

    // --- Coleções Customizadas ---
    public TCollection? MapCustomCollection<TCollection, T>(object? source)
        where TCollection : ICollection<T>, new()
        where T : class, new()
    {
        var items = Map<IEnumerable<T>>(source);
        if (items == null) return default;

        var collection = new TCollection();
        foreach (var item in items)
        {
            collection.Add(item);
        }
        return collection;
    }

    public TCollection? MapCustomCollection<TCollection, T>(
        object? source,
        Func<IEnumerable<T>, TCollection> factory
    ) where T : class, new()
    {
        var items = Map<IEnumerable<T>>(source);
        return items != null ? factory(items) : default;
    }

    // --- Métodos Assíncronos ---
    public async Task<T?> MapAsync<T>(object? source) where T : class, new()
    {
        if (source == null) return null;
        
        // Simula um mapeamento assíncrono (ex: chamadas a DB/API)
        return await Task.Run(() => Map<T>(source));
    }

    public async Task<IEnumerable<T>?> MapAsync<IEnumerable<T>>(
        object? source,
        int? maxDegreeOfParallelism = null
    ) where T : class, new()
    {
        if (source == null) return null;

        if (source is not IEnumerable<object> sourceCollection)
            throw new ArgumentException("O objeto de origem deve ser uma coleção.", nameof(source));

        var options = new ParallelOptions
        {
            MaxDegreeOfParallelism = maxDegreeOfParallelism ?? Environment.ProcessorCount
        };

        var results = new List<T>();
        await Parallel.ForEachAsync(
            sourceCollection,
            options,
            async (item, cancellationToken) =>
            {
                var mappedItem = await MapAsync<T>(item);
                if (mappedItem != null)
                {
                    lock (results) // Thread-safe para List<T>
                    {
                        results.Add(mappedItem);
                    }
                }
            }
        );

        return results;
    }

    public async Task<TCollection?> MapCustomCollectionAsync<TCollection, T>(
        object? source,
        Func<IEnumerable<T>, TCollection> factory,
        int? maxDegreeOfParallelism = null
    ) where T : class, new()
    {
        var items = await MapAsync<IEnumerable<T>>(source, maxDegreeOfParallelism);
        return items != null ? factory(items) : default;
    }
}

        => Map<ImmutableHashSet<T>>(source);

    public IImmutableQueue<T>? Map<IImmutableQueue<T>>(object? source) where T : class, new()
        => Map<ImmutableQueue<T>>(source);

    public IImmutableStack<T>? Map<IImmutableStack<T>>(object? source) where T : class, new()
        => Map<ImmutableStack<T>>(source);
}

Sintaxe correta para genéricos:

Adicionei todos os namespaces necessários no using

Restrições de tipo:

Mantive where T : class, new() consistentemente

Implementação Completa da Classe JMSAutoMapper
Aqui está a implementação completa da classe JMSAutoMapper com a interface IMapper corrigida, incluindo todos os métodos para coleções padrão, thread-safe, imutáveis e operações assíncronas.

csharp
using System;
using System.Collections;
using System.Collections.Generic;
using System.Collections.Concurrent;
using System.Collections.ObjectModel;
using System.Collections.Immutable;
using System.Linq;
using System.Runtime.Caching;
using System.Threading.Tasks;
using AutoMapper;

public class JMSAutoMapper : IMapper
{
    private readonly IMapper _autoMapper;
    private readonly MemoryCache _cache = MemoryCache.Default;

    public JMSAutoMapper(MapperConfiguration config)
    {
        _autoMapper = config.CreateMapper();
    }

    public JMSAutoMapper(IMapper mapper)
    {
        _autoMapper = mapper;
    }

    #region Métodos Base

    public T? Map<T>(object? source) where T : class, new()
    {
        if (source == null) return null;
        return _autoMapper.Map<T>(source);
    }

    private T? MapInternal<T>(object? source, HashSet<object> mappedObjects) where T : class, new()
    {
        if (source == null || mappedObjects.Contains(source))
            return null;

        mappedObjects.Add(source);
        return _autoMapper.Map<T>(source);
    }

    #endregion

    #region Coleções Padrão

    public IEnumerable<T>? MapIEnumerable<T>(object? source) where T : class, new()
    {
        if (source == null) return null;
        if (source is not IEnumerable enumerable)
            throw new ArgumentException("O objeto de origem deve ser uma coleção.", nameof(source));

        var mappedObjects = new HashSet<object>();
        return enumerable.Cast<object>()
            .Select(item => MapInternal<T>(item, mappedObjects))
            .Where(result => result != null)
            .ToList();
    }

    public List<T>? MapList<T>(object? source) where T : class, new()
    {
        var enumerable = MapIEnumerable<T>(source);
        return enumerable?.ToList();
    }

    public ICollection<T>? MapICollection<T>(object? source) where T : class, new()
    {
        return MapList<T>(source);
    }

    public IReadOnlyList<T>? MapIReadOnlyList<T>(object? source) where T : class, new()
    {
        return MapList<T>(source);
    }

    public IReadOnlyCollection<T>? MapIReadOnlyCollection<T>(object? source) where T : class, new()
    {
        return MapList<T>(source);
    }

    public T[]? MapArray<T>(object? source) where T : class, new()
    {
        var enumerable = MapIEnumerable<T>(source);
        return enumerable?.ToArray();
    }

    public HashSet<T>? MapHashSet<T>(object? source) where T : class, new()
    {
        var enumerable = MapIEnumerable<T>(source);
        return enumerable?.ToHashSet();
    }

    #endregion

    #region Dicionários

    public Dictionary<TKey, TValue>? MapDictionary<TKey, TValue>(object? source)
        where TKey : notnull
        where TValue : class, new()
    {
        if (source == null) return null;
        if (source is not IDictionary dictionary)
            throw new ArgumentException("O objeto de origem deve ser um dicionário.", nameof(source));

        var result = new Dictionary<TKey, TValue>();
        var mappedObjects = new HashSet<object>();

        foreach (DictionaryEntry entry in dictionary)
        {
            var key = MapInternal<TKey>(entry.Key, mappedObjects);
            var value = MapInternal<TValue>(entry.Value, mappedObjects);

            if (key != null && value != null)
            {
                result.Add(key, value);
            }
        }

        return result;
    }

    #endregion

    #region Coleções Thread-Safe

    public ConcurrentDictionary<TKey, TValue>? MapConcurrentDictionary<TKey, TValue>(object? source)
        where TKey : notnull
        where TValue : class, new()
    {
        var dictionary = MapDictionary<TKey, TValue>(source);
        return dictionary != null ? new ConcurrentDictionary<TKey, TValue>(dictionary) : null;
    }

    public BlockingCollection<T>? MapBlockingCollection<T>(object? source, int? boundedCapacity = null)
        where T : class, new()
    {
        var enumerable = MapIEnumerable<T>(source);
        if (enumerable == null) return null;

        return boundedCapacity.HasValue
            ? new BlockingCollection<T>(new ConcurrentQueue<T>(enumerable), boundedCapacity.Value)
            : new BlockingCollection<T>(new ConcurrentQueue<T>(enumerable));
    }

    #endregion

    #region Coleções Imutáveis

    public ImmutableList<T>? MapImmutableList<T>(object? source) where T : class, new()
    {
        var enumerable = MapIEnumerable<T>(source);
        return enumerable?.ToImmutableList();
    }

    public ImmutableDictionary<TKey, TValue>? MapImmutableDictionary<TKey, TValue>(object? source)
        where TKey : notnull
        where TValue : class, new()
    {
        var dictionary = MapDictionary<TKey, TValue>(source);
        return dictionary?.ToImmutableDictionary();
    }

    public ImmutableArray<T>? MapImmutableArray<T>(object? source) where T : class, new()
    {
        var enumerable = MapIEnumerable<T>(source);
        return enumerable?.ToImmutableArray();
    }

    public ImmutableQueue<T>? MapImmutableQueue<T>(object? source) where T : class, new()
    {
        var enumerable = MapIEnumerable<T>(source);
        return enumerable?.ToImmutableQueue();
    }

    public ImmutableStack<T>? MapImmutableStack<T>(object? source) where T : class, new()
    {
        var enumerable = MapIEnumerable<T>(source);
        return enumerable?.ToImmutableStack();
    }

    #endregion

    #region Métodos Assíncronos

    public async Task<T?> MapAsync<T>(object? source) where T : class, new()
    {
        if (source == null) return null;
        return await Task.Run(() => Map<T>(source));
    }

    public async Task<IEnumerable<T>?> MapIEnumerableAsync<T>(object? source, int? maxDegreeOfParallelism = null)
        where T : class, new()
    {
        if (source == null) return null;
        if (source is not IEnumerable enumerable)
            throw new ArgumentException("O objeto de origem deve ser uma coleção.", nameof(source));

        var options = new ParallelOptions
        {
            MaxDegreeOfParallelism = maxDegreeOfParallelism ?? Environment.ProcessorCount
        };

        var results = new List<T>();
        var sourceList = enumerable.Cast<object>().ToList();

        await Parallel.ForEachAsync(sourceList, options, async (item, cancellationToken) =>
        {
            var mappedItem = await MapAsync<T>(item);
            if (mappedItem != null)
            {
                lock (results)
                {
                    results.Add(mappedItem);
                }
            }
        });

        return results;
    }

    #endregion

    #region MemoryCache

    public CacheItem? MapToCacheItem<T>(object? source, string key, CacheItemPolicy? policy = null)
        where T : class, new()
    {
        if (source == null) return null;
        
        var mappedItem = Map<T>(source);
        return mappedItem != null 
            ? new CacheItem(key, mappedItem, policy: policy) 
            : null;
    }

    #endregion
}

A linha (MapInternal<TKey>(entry.Key, mappedObjects)) esta lançando a exceção CS0452
Resolução do Erro CS0452 na Implementação do JMSAutoMapper
O erro CS0452 ocorre quando tentamos usar um tipo genérico com restrição class em um contexto que não é compatível com essa restrição. No caso específico, está ocorrendo ao tentar mapear chaves de dicionário que podem ser tipos de valor (value types).

🔧 Solução para o Erro
Vamos corrigir a implementação do método MapDictionary para lidar adequadamente com os tipos de chave:

csharp
public Dictionary<TKey, TValue>? MapDictionary<TKey, TValue>(object? source)
    where TKey : notnull
    where TValue : class, new()
{
    if (source == null) return null;
    if (source is not IDictionary dictionary)
        throw new ArgumentException("O objeto de origem deve ser um dicionário.", nameof(source));

    var result = new Dictionary<TKey, TValue>();
    var mappedObjects = new HashSet<object>();

    foreach (DictionaryEntry entry in dictionary)
    {
        // Tratamento especial para a chave
        TKey key;
        if (entry.Key is TKey typedKey)
        {
            key = typedKey;
        }
        else
        {
            try
            {
                key = (TKey)Convert.ChangeType(entry.Key, typeof(TKey));
            }
            catch
            {
                continue; // Ignora entradas com chaves inválidas
            }
        }

        // Mapeamento do valor (que tem a restrição class)
        var value = MapInternal<TValue>(entry.Value, mappedObjects);
        if (value != null)
        {
            result.Add(key, value);
        }
    }

    return result;
}
📝 Explicação das Alterações
Problema Original:

Estávamos tentando usar MapInternal<TKey> onde TKey pode ser um tipo de valor

MapInternal tem restrição where T : class, new() que não permite tipos de valor

Solução Implementada:

Verificação direta de tipo (is TKey)

Conversão segura usando Convert.ChangeType

Ignora entradas com chaves inválidas em vez de lançar exceção

Manutenção de Segurança:

A restrição where TKey : notnull permanece

O tratamento de valores (que requerem class) continua inalterado

🚀 Implementação Atualizada do MapInternal
Para maior clareza, aqui está o método MapInternal atualizado:

private T? MapInternal<T>(object? source, HashSet<object> mappedObjects) where T : class, new()
{
    if (source == null || mappedObjects.Contains(source))
        return null;

    mappedObjects.Add(source);
    return _autoMapper.Map<T>(source);
}

[Fact]
public void MapToCacheItem_SemPolicy_Deve_Retornar_CacheItem()
{
    // Arrange
    var source = new ClienteOrigem { Id = 1, Nome = "Cliente Cache" };
    var cache = MemoryCache.Default;
    var cacheKey = "cliente_test_" + Guid.NewGuid();
    
    // Act
    var result = _mapper.MapToCacheItem<Cliente>(source, cacheKey);
    
    // Assert
    Assert.NotNull(result);

        Assert.Null(policy);
